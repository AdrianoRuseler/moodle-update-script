## Get scripts
```bash
mkdir scripts
cd scripts
wget https://raw.githubusercontent.com/AdrianoRuseler/moodledev-plugins/main/scripts/UpdateScripts.sh -O UpdateScripts.sh
chmod a+x UpdateScripts.sh
./UpdateScripts.sh
```
## Install phpMyAdmin
```bash
export LOCALSITENAME="pma"
export SITETYPE="PMA"
./CreateApacheLocalSite.sh
./InstallPMA.sh
```
